Short:Backup the files to tar.bz2 archieve and create MySQL DB dump.

Option:--separate
Create separated files for MySQL and the project

Option:--send-email=_address_
Create backup file and send it via mail on _address_

Option:--clean
With option --send-email remove backup files when mail has sended.

Option:--all
Create backup of the files to tar.bz2 archieve including all files.
