Стили шрифтов
**Жирный** Жирный Текст, обернутый в двойные * будет обрамлен HTML тегом <strong>
_Курсив_ Курсив Текст, обернутый в одинарные _ будет обрамлен HTML тегом <i> .
`Код` Код Чтобы выделить часть кода, необходимо обернуть текст в обратные кавычки(`).
>Цитата Цитата Markdown использует стиль почты > для выделения цитат.
Списки
* Элемент 1
* Элемент 2
  • Элемент 1
  • Элемент 2
 Для обозначения ненумерованных списков используют звездочки , плюсики и дефисы.
1. Элемент 1
2. Элемент 2
  1. Элемент 1
  2. Элемент 2
 Нумерованные списки обозначаются номером перед элементом

# Заголовок 1 #

 Стиль подчеркивания заголовков использует знак равенства (на первом уровне заголовков) и тире (на втором уровне заголовков). Заголовки 1-6 уровней обозначаются символом решетки, количество решеток соответствует уровню заголовка 1-6.
